# E-commerce

Bu proje, Internet Sunucu Programlama Dersi kapsamında geliştirilen "E-Ticaret" konulu web sitesidir. için yazılmış bir Node.js uygulamasıdır.

## Gereksinimler

- [Node.js](https://nodejs.org/) yüklü olmalıdır.

## Kurulum

1. Proje klasörünü bilgisayarınıza klonlayın:

    ```bash
    git clone https://github.com/kullaniciadi/proje-adı.git
    ```

2. Proje klasörüne gidin:

    ```bash
    cd proje-adı
    ```

3. Bağımlılıkları yükleyin:

    ```bash
    npm install
    ```

## Kullanım

Proje klasöründe aşağıdaki komutları kullanarak uygulamayı başlatabilirsiniz:

```bash
npm start
